Assumptions.

Only one grid is defined for each exploration.
A grid definition should be two numbers separate by a white space.

A grid should be defined before the first Rover.
Each Rover definition involves two lines. (Position and instructions)
The position of a Rover should have defined "X" and "Y" point and its facing separate by a white space.
The instructions of a Rover should be a combination of "L", "R" and "M".
The position of the Rover have to be into the grid definition parameters.

"N" numbers of Rovers can be defined. (press enter twice to start the exploration.)
